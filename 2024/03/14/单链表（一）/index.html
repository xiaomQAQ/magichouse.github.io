什么是单链表？

单链表（Singly Linked List）是一种常见的线性数据结构，它由一系列节点组成，每个节点包含两部分：数据和一个指向下一个节点的引用（指针或链接）。

单链表与顺序表的优缺点

顺序表：

优点：

可以随机存取，存储密度高。

缺点：

要求大片的连续空间，改变容量不方便。

单链表：

优点：

不要求大片连续空间，改变容量方便。

缺点：

不可以随机存取，要耗费一定空间存放指针。

单链表代码部分

定义单链表

typedef struct LNode {//定义单链表结点类型
	int data;//每个结点存放一个数据元素
	struct LNode* next;//指针指向下一个结点
}LNode,*LinkList;

typedef关键字：数据类型重命名
typedef <数据类型> <别名> 例如本例中的：typedef struct LNode {…} LNode, * LinkList;相当于 typedef struct LNode LNode;typedef struct LNode * LinkList;这两句。

单链表初始化

不带头结点的单链表

//初始化单链表(不带头结点)
bool InitList(LinkList& L) {
	L = NULL;//空表，暂时没有结点。防止之前的脏数据
	return true;
}

//判断单链表是否为空（不带头结点）
bool Empty(LinkList L) {
	if (L == NULL)
		return true;
	else
		return false;//直接return(L=NULL);也可以
}

void test() {
	LinkList L;//声明一个指向单链表的指针，此处并没有创建一个结点。
	//初始化一个空表
	InitList(L);
 
}

判断不带头结点的单链表是否为空只需要判断头指针是否为空。

带头结点的单链表

//初始化单链表（带头结点）
bool InitList1(LinkList& L) {
	L = (LNode*)malloc(sizeof(LNode));//分配一个头结点
	if (L = NULL) {//内存不足，分配失败
		return false;
	}
	L->next = NULL;//头结点之后暂时没有结点
	return true;
}
//判断单链表是否为空（带头结点）
bool Empty1(LinkList L) {
	if (L->next == NULL)
		return true;
	else
		return false;
}

刚创建的头结点是空的，不存储任何数据。
